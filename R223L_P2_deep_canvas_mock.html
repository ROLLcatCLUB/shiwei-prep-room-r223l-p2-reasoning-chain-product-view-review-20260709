1. 资料证据层：先看系统依据什么推

证据先于页面
大观念
体验传统工艺，带着好奇心和想象力设计改进文具，从而改善学习生活。
基本问题
怎样全方面了解文具，并感受它的重要性？
表现性任务
形成《一年级购买文具建议书》和《文具课堂使用指南》。

大观念

文具不是静物对象，而是学习生活里的工具、设计和文化载体。

基本问题

学生要从观察、使用、故事和改造中理解文具为什么重要。

表现性任务

用建议书、指南、代言展示证明理解，而不只交一件作品。

四阶段学习责任

初探、再探、智造、结交分别承担发现、理解、改造、迁移。

评价证据链

观察记录、设计理由、作品说明、辩论表达和指南质量共同成证据。

2. 单元 DNA 判断

主类型：设计应用 + 生活问题解决
辅助类型：观察表达、工艺技术文化理解、公开表达任务
不是：单纯画文具、普通手工、泛泛介绍物品

3. 学生问题推理

  • 一年级观察容易散，所以需要摸、看、圈、说。
  • 注意力短，所以活动要短回合、游戏化、可操作。
  • 手工能力弱，所以先 1+1 支架，再 1+n 变化。
  • 表达需要句式支架，所以要学习单、词库和代言话术。
  • 好奇心强，所以可以用圆珠笔、太空笔、毛笔工艺打开价值线。

4. 四阶段推理链

  • 初探：认识、触摸、描述、代言、拼搭。
  • 再探：剪影、圆珠笔、太空笔、毛笔工艺、智造交流。
  • 智造：结构观察、1+1 合作设计、1+n 文具大变身、笔友汇。
  • 结交：线上交流、小型辩论、建议书、使用指南。
高风险提醒：本页是推理链产品视图，不授权正式 UI、R97B 路由、模型生成、数据库写入或正式应用。

5. 当前课时深链：第三阶段「智造·新朋友」

这一阶段不是“让学生随便改造文具”，而是把前两阶段的观察、故事和价值理解，转化为低门槛的设计改进、创作表达和可评价证据。

当前唯一展开主区：从理解文具到改进文具

前置证据来自初探的观察描述、再探的工艺和智造故事。本课责任是让学生说出一个使用问题，选择材料或技法完成一次文具改造，并能解释改造理由。

教师怎么说

“刚才我们认识了文具的结构和故事。现在请你想一想：这个文具在使用时有没有一个小麻烦？你能不能给它交一个新朋友，让它更好用？”

学生怎么做

观察结构，提出问题，先用 1+1 组合试一试，再从 1+n 的材料和技法中选择一种完成改造。

素材 / 大屏 / 学习单

大屏显示结构图、问题例子、材料菜单和技法菜单；学习单记录“原文具、问题、新朋友、改造理由”。

评价证据

保留改造草图或作品照片，能说出使用问题、材料选择、改造理由和同伴建议。

1. 结构观察
为什么存在学生容易直接装饰，先看功能结构。
课堂动作圈出握、写、收纳、保护等关键位置。
证据结构标注或口头说明。
2. 1+1 合作设计
为什么存在降低设计难度，用一个材料朋友解决一个小问题。
课堂动作铅笔 + 粘土、橡皮 + 纸套等低门槛组合。
证据能说明“我解决了什么问题”。
3. 1+n 大变身
为什么存在把单一试想到多材料、多技法选择。
课堂动作从纸材、线材、形状材料和技法菜单中选择。
证据作品 + 材料选择理由。
4. 笔友汇
为什么存在把个人作品变成公共表达和建议。
课堂动作同伴看一看，说一个优点和一个使用建议。
证据同伴评价、推荐语、使用提醒。

6. 组件调用逻辑：组件不是工具按钮，是解决学生问题的教学动作

圈一圈

解决：学生说不清依据。

这里用：结构观察时标出关键位置。

证据：标注图或关键词。

技法菜单

解决：学生有想法但不会做。

这里用：1+n 文具大变身选择路径。

证据：技法选择和理由。

学习单记录

解决：过程证据容易丢。

这里用：记录原问题、新朋友和改造理由。

证据：学习单行或作品说明。
